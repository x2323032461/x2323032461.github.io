hexo指令

发表于 2021-02-21 更新于 2021-02-22

​

创建文章

​ hexo new “文章名称”

​ 或

​ hexo n “文章名称”

清理页面

hexo clean

生成页面

​ hexo g

本地预览页面（show）

​ hexo s

部署发布

​ hexo d

生成标签页面 页面文章头部需要添加 type: tags

文章通过添加（tags: 标签）来添加标签

​ hexo new page 标签

生成分类页面 头部需要添加 type: categoriese

文章通过添加（categoriese: 分类）来添加分类

​ hexo new page 分类
