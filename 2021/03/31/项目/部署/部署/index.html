修改项目名

启动jar包

java -jar *.jar

设置日志文件位置

java -jar business.jar -> log/project.log

后台启动（此时当服务器连接关闭，启动的项目也会关闭）

java -jar business.jar -> log/project.log &

服务器关闭项目依旧运行

nohup java -jar business.jar -> log/project.log &

限制运行大小

java -jar -Xms256m -Xmx521m
